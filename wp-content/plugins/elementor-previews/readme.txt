Adds thumbnail preview functionality to Elementor's template library.

== Description ==

Elementor Template Previews enhances the Elementor template library by adding visual thumbnail previews. This makes it easier to browse and select templates directly from the Elementor editor.

**Features:**

* Visual thumbnail previews in the template library
* Public preview URLs for templates
* Seamless integration with Elementor editor
* Lightweight and performant

**Requirements:**

* WordPress 5.0 or higher
* Elementor plugin (free or Pro)
* PHP 7.4 or higher

== Installation ==

1. Upload the `elementor-previews` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Permalinks and click "Save Changes" to flush rewrite rules
4. Open the Elementor editor and access the template library

== Frequently Asked Questions ==

= Does this work with Elementor Free? =

Yes, this plugin works with both Elementor Free and Elementor Pro.

= How do I access the template previews? =

Open any page with Elementor, click the folder icon to open the template library, and you'll see the enhanced preview functionality.
